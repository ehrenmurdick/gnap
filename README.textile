h1. Gnap
<sub>%{font-size:15px;font-weight:normal}/n&#230;p/%</sub>

Gnap is an interface to Gnip's api. It has not been written yet. The following code is how I hope to make it work, and is subject to change.

<pre>
<code>
require 'gnap'
gnap = Gnap.new("username", "password")

# current activity for twitter
gnap.publishers("twitter").activity

# last five minutes
gnap.publishers("twitter").activity(5.minutes.ago..Time.now)

# last hour
gnap.publishers("twitter").activity(1.hour.ago..Time.now)

# from a filter
gnap.publishers("twitter").filters("bob").activity


# create a new filter
gnap.publishers("twitter").filters.new(:actor => "some actor rule text")
</code>
</pre>
